# Encriptador y Desencriptador de Texto

Este proyecto es un encriptador y desencriptador de texto desarrollado como proyecto final para el proeceso de selección del programa ONE de Oracle y ALura. 
Permite a los usuarios ingresar texto y convertirlo en un formato encriptado, así como revertir la encriptación para obtener el texto original.

## Funcionalidades

- **Encriptar texto**: Convierte letras específicas en combinaciones de letras, haciendo que el texto sea irreconocible.
- **Desencriptar texto**: Revierte el proceso de encriptación, recuperando el texto original.
- **Copiar al portapapeles**: Permite al usuario copiar el texto encriptado o desencriptado fácilmente.
- **Limpiar el texto**: Opción para limpiar el área de texto de entrada y salida.
- **Interfaz amigable**: Diseño sencillo y fácil de usar.

## Tecnologías Utilizadas

- **HTML**: Estructura del documento.
- **CSS**: Estilos para el diseño visual.
- **JavaScript**: Funcionalidades de encriptación, desencriptación y manipulación del DOM.
